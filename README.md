# Перевірка пароля

В класі ***Main*** методом *checkPassword* виповнюється перевірка, чи задовільняє пароль цим умовам:
- Довжина 8..16
- Є хоча б 1 цифра
- Є хоча б 1 велика літера (A-Z)
- Є хоча б 1 спецсимвол із набору !@#$%
- Не починається з цифри

Якщо будь-що не виконано → **WEAK**, інакше - **STRONG**
